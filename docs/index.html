🎙 GroqVoice

Hold a key, speak, release — your speech is transcribed (Whisper via Groq) and pasted into the focused window. Say “task: …” / «задача: …» and the LLM answer is pasted instead. Russian + English out of the box.

🪟 Windows

⬇ GroqVoice-fd.exe — 720 KB ⬇ GroqVoice-sc.exe — 70 MB, self-contained

The small build needs the .NET 8 Desktop Runtime. SmartScreen warning? More info → Run anyway (not code-signed yet). Hotkey: hold Win+Ctrl.

🍎 macOS (Apple Silicon + Intel, macOS 13+)

Paste in Terminal — downloads, installs and launches:

curl -fsSL https://raw.githubusercontent.com/abirzgals/GroqVoice/main/macos/install.sh | bash
⬇ or GroqVoice-mac.zip — 200 KB

First launch:

  1. Allow Microphone when asked
  2. Allow Accessibility (the prompt opens System Settings — toggle GroqVoice on)
  3. Paste your free Groq API key

Hotkey: hold Fn (🌐). If you download the zip manually, run xattr -cr GroqVoice.app after unzipping (the app isn't notarized).
